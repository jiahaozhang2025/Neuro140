# Neuro140
Neuro140 final project codes

Extension work of: St-Yves, G., Allen, E. J., Wu, Y., Kay, K. & Naselaris, T. Brain-optimized deep neural network models of human visual areas learn non-hierarchical representations. Nat. Commun. 14, 3329 (2023).

Dataset: http://naturalscenesdataset.org/

Dataset Paper: Allen, St-Yves, Wu, Breedlove, Prince, Dowdle, Nau, Caron, Pestilli, Charest, Hutchinson, Naselaris*, & Kay*. A massive 7T fMRI dataset to bridge cognitive neuroscience and artificial intelligence. Nature Neuroscience (2022).

Original codes: https://github.com/styvesg/nsd_gnet8x.git.

This repository contains new codes modified from the original codes. So it requires merging with the original codes to be able to run.

| Notebook | Description |
|------|-------------|
|`alexnet_fwrf_encoding_multisubjects_1`| original AlexNet model training|
|`alexnet_flexible_encoding_multisubjects_x`| variant AlexNet model training (x for different training schemes)|
|`gnet8j_encoding_multisubjects_1_all`| original GNet8j model training|
|`gnet8j_encoding_multisubjects_3_all`| variant GNet8j model training|
|`gnet8r_encoding_multisubjects_1_all`| original GNet8r model training|
|`gnet8r_encoding_multisubjects_3_all`| variant GNet8r model training|
|`gnet8x_prediction_multisubjects_all`| GNet8j and GNet8r prediction (unused)|
|`gnet8x_tuning_analysis_and_accuracy_1_all_eval`| original GNet8j and GNet8r tuning analysis|
|`gnet8x_tuning_analysis_and_accuracy_1_all_plot`| plots of original GNet8j and GNet8r tuning analysis results|
|`gnet8x_tuning_analysis_and_accuracy_3_all_eval`| variant GNet8j and GNet8r tuning analysis|
|`gnet8x_tuning_analysis_and_accuracy_3_all_plot`| plots of variant GNet8j and GNet8r tuning analysis results|
